## 📊 Прайс-лист и интеграция с Google Sheets

### Как работает система прайса
- **Источник**: Google Таблица (публичный доступ через `gviz/tq`)
- **URL шаблон**:  
  `https://docs.google.com/spreadsheets/d/{SHEET_ID}/gviz/tq?tqx=out:json`
- **Преобразование**:  
  Скрипт (например, `scripts/update-price.js`) или ручной экспорт → `price.json` в `/site/public/`
- **Формат `price.json`**:
  ```json
  {
    "categories": [
      {
        "id": "phones",
        "name": "Смартфоны и планшеты",
        "services": [
          {
            "id": "screen-replace",
            "name": "Замена дисплея",
            "price": 1200,
            "time": "20 мин",
            "category": "phones"
          }
        ]
      }
    ],
    "allServices": [...] // плоский список для поиска
  }
  ```

### Поиск по прайсу
- На `price.html`: используется `debounced` поиск по `allServices`
- Фильтрация по категориям через `data-category`
- Подсветка совпадений через `<mark>`
- Работает полностью клиентски (без сервера)

### Обновление прайса
- **Рекомендуемый способ**:  
  1. Изменить Google Таблицу  
  2. Запустить скрипт (например, `node scripts/update-price.js`)  
  3. Сгенерировать новый `price.json`  
  4. Закоммитить в Git
- **Альтернатива**: ручной экспорт в JSON через Google Apps Script или сторонние утилиты

### Безопасность и надёжность
- Используется **публичный URL Google Sheets** — без API-ключа (требует публичного доступа к таблице)
- Fallback: если `price.json` не загружается — показывается сообщение «Прайс временно недоступен»
- Кэширование: `price.json` кэшируется браузером (можно добавить `Cache-Control: max-age=3600`)
