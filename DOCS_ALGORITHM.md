# 🤖 Алгоритм «Контент-Конвейер Админ.Ко 1000»

Этот документ описывает полный цикл автономной генерации SEO-контента для сайта «Админ.Ко». Система работает без участия человека, обеспечивая непрерывный рост поискового трафика.

---

## 🏗️ Архитектура системы

Система состоит из 5 независимых модулей, объединенных мастер-скриптом.

### 1. Триггер: `scripts/cron-job.sh`
*   **Функция:** Точка входа для системного планировщика (Cron).
*   **Частота:** Каждые 3 часа (8 раз в сутки).
*   **Безопасность:** Использует абсолютные пути и логирует ошибки в `logs/cron.log`.
*   **Действие:** Настраивает окружение Node.js и запускает Оркестратор.

### 2. Оркестратор: `scripts/content-orchestrator.js`
*   **Функция:** Управление очередью и ресурсами.
*   **Шаги:**
    1.  **Проверка тем:** Проверяет `scripts/prioritized_topics.json`. Если тем < 20, запускает Анализатор Ключевых Слов.
    2.  **Распределение:** Выбирает 16 тем (по 2 из каждой из 8 категорий: Phone, Laptop, B2B, CCTV, Consoles, Appliances, TV, Data Recovery).
    3.  **Запуск генерации:** Передает каждую тему по одной в Авто-блогер.
    4.  **Завершение:** Обновляет индекс блога и запускает Синхронизацию UI.

### 3. Анализатор (AI-Стратег): `scripts/ai-keyword-analyzer.js`
*   **Функция:** Превращение сырых данных в осмысленные темы.
*   **Вход:** Список ключевых слов из `scripts/real_keywords.json`.
*   **Интеллект:** Сверяется с уже написанными статьями (`site/public/blog/index.json`), чтобы избежать дублей.
*   **Выход:** Сгенерированные через Gemini 2.5 Flash темы с прописанными SEO-ключами.

### 4. Производство: `scripts/auto-blogger.js`
*   **Функция:** Написание и верстка статьи.
*   **Технология:** Gemini 2.5 Flash.
*   **Этапы:**
    1.  Генерация мета-данных (Title/Description).
    2.  Составление плана (H2-H4 заголовки).
    3.  Поэтапная генерация текста глав (для обхода лимитов токенов).
    4.  Финальная сборка HTML с использованием дизайн-системы **Stitch**.
    5.  Сохранение файла в `site/public/blog/[slug].html`.

### 5. Логистика и Индексация:
*   `scripts/reindex-blog-safe.js`: Сканирует папку блога, извлекает дату публикации и мета-теги, обновляет общий реестр `site/public/blog/index.json`.
*   `scripts/sync-ui.js`: Обновляет хедер, футер и блоки перелинковки во всех HTML-файлах сайта, обеспечивая видимость новых статей.

---

## 🛠️ Мониторинг и Отладка

*   **Логи выполнения:** `logs/cron.log` (все системные сообщения).
*   **Статус тем:** `scripts/prioritized_topics.json` (список задач на будущее).
*   **База статей:** `site/public/blog/index.json` (реестр всего опубликованного контента).

---

## 📜 Правила «Конституции» (GEMINI.md)
1.  **Никакого Markdown в HTML:** Только валидные теги.
2.  **Сохранение структуры:** Не менять контент между навигацией и футером в статьях.
3.  **Эталонность:** Все статьи должны соответствовать дизайну `phones.html`.
