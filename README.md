## 📥 Обновление прайс-листа из Google Sheets

### Как устроена система
- Прайс хранится в Google Таблице (публичный доступ)
- Скрипт генерирует `price.json` → используется на `price.html`
- Поиск и фильтрация — клиентские (без сервера)

### Шаги для обновления
1. **Измените Google Таблицу** (добавьте/измените услуги, цены, категории)
2. **Сгенерируйте новый `price.json`**:
   - Вариант A (рекомендуется): запустите скрипт (если есть `scripts/update-price.js`):
     ```bash
     node scripts/update-price.js
     ```
   - Вариант B (ручной): экспортируйте таблицу как JSON через:
     - Google Apps Script (автоматизация)
     - или онлайн-конвертер: [sheets-to-json.net](https://sheets-to-json.net)
3. **Проверьте структуру `price.json`**:
   - Должен содержать `categories` и `allServices`
   - Пример:
     ```json
     {
       "categories": [{ "id": "phones", "name": "Смартфоны", "services": [...] }],
       "allServices": [...]
     }
     ```
4. **Закоммитьте изменения**:
   ```bash
   git add price.json && git commit -m "price: обновлено от 25.02.2026"
   ```

> ⚠️ Если `price.json` не загружается — на `price.html` появится сообщение: *«Прайс временно недоступен»*.

## 🔐 Безопасность и рекомендации

| Тема | Статус | Рекомендация |
|------|--------|--------------|
| Google Sheets доступ | Публичный URL | Для конфиденциальности: используйте Service Account + API ключ |
| CORS | Нет проблем (статика) | Не требуется прокси |
| Popup-blocker | Есть риск | Добавьте fallback: при ошибке — показать копируемую ссылку |
| Кэширование | Нет `Cache-Control` | Добавьте в `.htaccess`: `Header set Cache-Control "max-age=3600"` |
| Доступность (a11y) | Частично | Добавьте `aria-modal="true"`, управление фокусом в модалке |

## 📦 Деплой на хостинг

### Варианты:
| Хостинг | Как развернуть |
|---------|----------------|
| Netlify / Vercel | Загрузите папку `site/public` как root |
