# 🤖 Алгоритм «Контент-Конвейер Админ.Ко 1000»

Этот документ описывает полный цикл автономной генерации SEO-контента для сайта «Админ.Ко».

---

## 🏗️ Архитектура системы

1. **Триггер (Cron):** `scripts/cron-job.sh` запускается каждые 3 часа.
2. **Оркестратор:** `scripts/content-orchestrator.js` управляет очередью и делает бэкап.
3. **Бэкап-менеджер:** `scripts/backup-manager.js` сохраняет 10 последних версий блога перед изменениями.
4. **Производство:** `scripts/auto-blogger.js` генерирует экспертный HTML-контент через Gemini 2.5 Flash.
5. **Индексация:** `scripts/reindex-blog-safe.js` обновляет `index.json` с учетом дат публикации.
6. **Синхронизация:** `scripts/sync-ui.js` прокидывает единый хедер и футер во все файлы.

---

## 📜 Стандарты качества (Constitution)

- **Дизайн:** Использование системы Stitch, Tailwind v3.4, поддержка темной темы.
- **Чистота:** Категорический запрет на Markdown (`**`, `#`, `-`). Только валидный HTML.
- **UX:** Автоматическая разбивка на абзацы, стильные блоки цен и инженерных нюансов.
- **SEO:** Уникальные мета-теги, JSON-LD разметка, правильные заголовки.
- **Единство:** Карта вставляется только ОДИН РАЗ из футера.

---

## 🛠️ Команды управления

- `node scripts/content-orchestrator.js` — запустить полный цикл генерации (16 статей).
- `node scripts/restore-articles.js` — восстановить битые статьи из индекса.
- `node scripts/sync-ui.js` — обновить шапку и подвал на всём сайте.
- `node scripts/reindex-blog-safe.js` — обновить список статей в блоге.
